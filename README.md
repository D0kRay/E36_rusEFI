## English

Small community project for a rusEFI **MS41 port** (BMW E36 / M52).
This repository contains a working TunerStudio project with tune, dashboards, and configuration files.

### Pictures

> Pictures will be added later.

### Requirements

- rusEFI firmware (matching your hardware)
- **TunerStudio**
- This repository (including INI files)

### TunerStudio download

- Official download page: [https://www.tunerstudio.com/index.php/downloads](https://www.tunerstudio.com/index.php/downloads)
- Product page: [https://www.tunerstudio.com/](https://www.tunerstudio.com/)

### MegaLogViewer (log analysis)

- Product page/download: [https://www.efianalytics.com/MegaLogViewer/](https://www.efianalytics.com/MegaLogViewer/)

### Required INI files (needed for correct TunerStudio operation)

The INI files are required for this project. Please use:

- Main INI: [rusefi_mega100.ini](rusefi_mega100.ini)

### Quick start

1. Download/clone this repository.
2. Create a new project in TunerStudio.
3. Select [rusefi_mega100.ini](rusefi_mega100.ini) as the INI file (or the INI referenced by your project setup).
4. Optionally load an existing tune: [CurrentTune.msq](CurrentTune.msq).
5. Connect ECU and verify sensor/pin configuration.
